# 📝 To-Do List App

A simple and responsive To-Do List web application built with **HTML, CSS, and JavaScript**.  
This project allows users to add, mark, delete, and clear tasks with a clean UI.

---

## 🚀 Features
- Add new tasks with a single click
- Mark tasks as completed (strike-through effect)
- Delete individual tasks
- Clear all tasks at once
- Responsive design (works on mobile, tablet, and desktop)

---

## 🎨 Tech Stack
- **HTML5** – structure
- **CSS3** – styling and responsiveness
- **JavaScript (ES6)** – interactivity and DOM manipulation
